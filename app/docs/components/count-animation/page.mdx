# Count Animation

The count animation component is designed to create a captivating user experience by animating numerical values as they increase or decrease. This effect draws the user's attention to important data, making the information more dynamic and engaging. Whether it's displaying statistics, financial figures, or progress counters, the smooth animation of numbers adds a sense of liveliness and interactivity to your interface.

### API

| Prop      | Type     | Default     | Description                                |
|-----------|----------|-------------|--------------------------------------------|
| number    | `number` |             | The target number to animate towards       |
| className | `string` | `undefined` | Optional CSS class for styling             |
| prefix    | `string` | `undefined` | Text or symbol displayed before the number |
| suffix    | `string` | `undefined` | Text or symbol displayed after the number  |
